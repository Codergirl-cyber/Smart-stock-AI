# SellerSync Demo Script

This demo is designed to show the app as a polished AI Inventory OS in 3–5 minutes.

## Preparation

1. Seed demo content:
   - `npm run demo-data`
   - enable demo mode in your environment: `VITE_DEMO_MODE=true`
2. Start the app:
   - `npm run dev`
3. Login as the demo user or use authenticated demo access.

## 1. Dashboard Overview (30s)

- Open the `Dashboard`.
- Call out the top KPI cards:
  - Revenue
  - Orders
  - Products
  - Pending shipments
- Mention the AI Insights summary and how it combines inventory health, trend, and restock risk.

## 2. AI Restock Recommendations (30s)

- Show the `AI Restock Recommendations` card.
- Highlight recommended reorder quantities and priority badges.
- Emphasize how the system flags SKUs that need restock now.

## 3. Demand Forecasting (30s)

- Show the `7-Day Sales` chart.
- Explain the trend line and weekend sales behavior.
- Mention how the forecast feeds the restock recommendations and avoids stockouts.

## 4. Inventory Copilot (30s)

- Open `Inventory Copilot`.
- Ask a pre-defined question or click a suggestion.
- Show the generated answer and how it uses inventory context to provide actionable recommendations.

## 5. Executive AI Report (30s)

- Show the `AI Business Analyst` panel.
- Point out the executive summary, inventory health score, and top recommendations.
- Mention export buttons for JSON/CSV.

## 6. Agent Activity Feed (30s)

- Open the `AI Agent Activity` feed.
- Highlight automated inventory tasks and execution logs.
- Mention how agents continually monitor stock and demand.

## 7. Automated Inventory Management (30s)

- Show the `Executive Command Center`.
- Explain how it runs inventory analysis and restock planning.
- Mention the fallback demo mode if AI keys are unavailable.

## Closing (15s)

- Reiterate that SellerSync combines real inventory data with AI automation.
- Emphasize reliability: demo mode, fallback content, and a polished dashboard.
- End with a simple value statement: "This is the future of inventory operations for modern sellers."
